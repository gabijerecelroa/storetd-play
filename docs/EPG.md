# EPG / XMLTV

## MVP

El MVP no procesa XMLTV todavia. La estructura queda preparada para version Pro.

## Version Pro

Funciones:

- Configurar URL XMLTV por cliente.
- Descargar periodicamente con WorkManager/backend worker.
- Asociar `tvg-id` de M3U con `channel id` de XMLTV.
- Mostrar programa actual y proximo.
- Guia tipo TV.
- Busqueda por programa.
