# Recomendaciones legales

Este proyecto no incluye contenido audiovisual ni listas IPTV.

## Principios

1. No incluir listas preinstaladas.
2. No incluir contenido de terceros.
3. No usar logos o nombres de canales sin autorizacion comercial.
4. Exigir que el cliente tenga derechos sobre las listas.
5. Tener terminos y condiciones.
6. Tener politica de privacidad.
7. Retirar listas o canales reportados por derechos.
8. No promocionar contenido premium sin licencia.
9. No mostrar URLs sensibles al usuario final.
10. Documentar el uso permitido.

## Texto sugerido

"Esta aplicacion funciona exclusivamente como reproductor de contenido autorizado. El proveedor de la aplicacion no distribuye contenido audiovisual no autorizado ni garantiza acceso a canales de terceros."
