# Seguridad

## No hacer

- No subir keystore.
- No subir claves API.
- No guardar contrasenas en texto plano.
- No imprimir tokens en logs.
- No mostrar URLs completas al usuario.
- No incluir listas ilegales.

## Hacer

- HTTPS obligatorio.
- JWT con expiracion.
- Refresh token seguro.
- Almacenamiento seguro en Android.
- Secrets en GitHub.
- Logs sanitizados.
- Revocacion de dispositivos.
- Rate limiting en backend.
- Backups cifrados.
